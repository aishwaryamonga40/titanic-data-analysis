This project analyzes the Titanic dataset to explore factors that influenced passenger survival.
The analysis shows that female passengers had a significantly higher survival rate than male passengers, with 233 female survivors compared to 109 male survivors.
The data also suggests that passenger class influenced survival, with first-class passengers having the highest number of survivors.
Pivot tables were used to examine survival patterns by gender and passenger class, and the results were visualized using pie charts and bar graphs in Microsoft Excel.
Tools used: Microsoft Excel, Pivot Tables, Data Visualization.
